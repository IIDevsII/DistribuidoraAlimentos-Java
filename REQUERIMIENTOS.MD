# Requerimientos del Proyecto

## Requerimientos Funcionales
1. Permitir al usuario ingresar:
   - Marca
   - Modelo
   - Cilindrada
   - Tipo de combustible
   - Capacidad de pasajeros
2. Mostrar los datos ingresados en pantalla.
3. Calcular el costo de despacho según reglas de negocio.
4. Permitir registro de usuario con Gmail (simulado).
5. Ejecutarse en JVM en dispositivos móviles/PC.

## Requerimientos No Funcionales
1. Código comentado línea por línea.
2. Debe compilarse y ejecutarse sin IDE (solo línea de comandos).
3. Uso de `Scanner(System.in)` y `System.out`.
4. Documentación completa organizada en GitHub.
5. Cronograma inicial reflejado en GitHub Projects.
